# 👶 新手爸爸育儿指南

一个专为新手爸爸打造的 0-12 个月婴儿成长护理指南应用，包含详细的发育指标、护理建议、营养提示、成长里程碑等内容。

## ✨ 功能特点

- 📅 **月龄选择**：快速查看 0-12 个月任意阶段的育儿指南
- 📊 **发育指标**：详细的身体发育、认知能力、运动能力数据
- 👨‍👦 **护理指南**：睡眠、洗澡、换尿布等日常护理技巧
- 🍼 **喂养建议**：母乳、配方奶的科学喂养方法
- 🏆 **成长里程碑**：记录和追踪宝宝的成长重要时刻
- 📹 **视频教程**：图文并茂的护理教学资源
- ⚠️ **安全提醒**：每个阶段的安全注意事项
- ❓ **常见问题**：新生儿黄疸、肠绞痛等问题的解决方案
- 📈 **成长时间轴**：可视化展示宝宝成长历程
- ✅ **成长清单**：记录并导出宝宝成长数据
- 💉 **疫苗接种提醒**：疫苗接种进度跟踪和提醒
- 📊 **成长曲线图表**：WHO标准参考的成长数据记录
- 📔 **育儿日记**：记录宝宝成长的点点滴滴

## 🛠️ 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **TypeScript** - 类型安全
- **Vite** - 极速开发构建工具
- **Vue Router** - 官方路由管理
- **Pinia** - 状态管理
- **Element Plus** - UI 组件库
- **Axios** - HTTP 客户端（预留接口调用）

## 📱 功能页面

### 首页
- 宝宝信息设置（昵称、生日、性别）
- 当前月龄快速概览
- 月龄网格选择
- 快捷功能入口（疫苗接种、成长曲线、育儿日记）

### 月龄详情页
- 发育指标：体重、身高、头围等数据
- 护理指南：睡眠、洗澡、换尿布等
- 喂养建议：母乳/配方奶的科学喂养
- 成长里程碑：可勾选记录
- 常见问题：症状、解决方案、就医指导
- 安全提醒：该月龄需要注意的安全事项
- 学习资源：视频教程和图片图解

### 成长时间轴
- 0-12 个月完整发育历程展示
- 里程碑完成状态可视化
- 快速跳转到各月详情

### 成长清单
- 所有里程碑的管理和勾选
- 完成率统计
- 导出成长记录（JSON 格式）
- 重置功能

### 💉 疫苗接种提醒
- 接种进度概览（总数、已接种、待接种、完成率）
- 即将接种提醒（根据宝宝月龄智能提醒）
- 疫苗接种时间表（按月龄分组）
- 疫苗筛选（全部/已接种/待接种）
- 免费/自费疫苗标识
- 接种记录管理（标记完成、记录日期）

### 📊 成长曲线图表
- 当前数据卡片（体重、身高、头围）
- 成长趋势图表（柱状图展示）
- 多维度切换（体重/身高/头围）
- 成长记录管理（添加、编辑、删除）
- WHO 参考标准对照表（0-12月龄男女宝宝）

### 📔 育儿日记
- 统计概览（日记数、照片数、精选数）
- 写日记功能（日期、月龄、心情、标题、内容）
- 照片上传和预览（支持多图）
- 标签系统（快捷标签 + 自定义标签）
- 搜索和筛选（按内容搜索、按心情筛选）
- 收藏功能（精选日记）
- 完整的CRUD操作

## 💾 数据持久化

应用使用 `localStorage` 存储以下数据：
- 宝宝基本信息（昵称、生日、性别）
- 已完成的成长里程碑
- 疫苗接种记录
- 成长数据记录（体重、身高、头围）
- 育儿日记
